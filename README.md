# EncodeLab

EncodeLab is a small web-based project for learning and visualizing data encoding techniques through a multi-layer encoding pipeline.

Users can combine multiple encoders, reorder layers, and view step-by-step transformations of the encoded data.

## Features

- Multi-layer encoding pipeline
- Step-by-step visualization
- Interactive UI
- Multiple encoding methods

## Supported Encoders

- ASCII
- Binary
- Hex
- ROT13
- ROT47
- Base32
- Base45
- Base58
- Base62
- Base64
- Base85

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open `index.html` in your browser  
or use Live Server in VS Code.

## Purpose

This project was created for educational purposes to help beginners understand how different encoding systems work.
